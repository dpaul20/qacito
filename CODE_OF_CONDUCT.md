# Code of Conduct

## Our Commitment

We are committed to a respectful, harassment-free experience for everyone.

## Expected Behavior

1. Be respectful and constructive.
2. Assume good intent, ask questions before judging.
3. Focus feedback on code and decisions, not people.
4. Welcome newcomers and different viewpoints.

## Unacceptable Behavior

1. Harassment, discrimination, or personal attacks.
2. Doxxing or sharing private information.
3. Threats, intimidation, or abusive language.
4. Disruptive behavior that blocks collaboration.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this policy.
Violations may result in warnings, temporary restrictions, or removal from project spaces.
